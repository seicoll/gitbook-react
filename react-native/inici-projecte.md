# Creació del proejecte React Native amb Vite i Nativewind

## Crear el projecte Expo i React Native

- [Documentació Expo](https://docs.expo.dev/more/create-expo/)

### Creem el projecte amb Vite

```bash
npm create vite@latest my-react-native-app -- --template blank-typescript
```

### Accedim al directori del projecte

```bash
cd my-react-native-app
```

### Instal·lem les dependències

```bash
npm install
```

### Iniciem el projecte

```bash
npm run start
```
